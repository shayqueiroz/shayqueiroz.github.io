Neste site você encontrará, de forma objetiva e sintética, os destaques dos conteúdos aprendidos ao longo de 4 módulos do curso de HTML5 e CSS3 ministrados pelo professor Gustavo Guanabara. As páginas de cada módulo apresentam diagramação diferente, pois foram desenvolvidas com base no que foi aprendido em cada módulo de forma individual. Assim, é possível ver a evolução do desenvolvimento do layout a medida em que se aprende mais e mais funções para configurações de estilo CSS e semântica HTML.
O objetivo deste site é sintetizar conteúdos de HTML e CSS de forma simples para ajudar no entendimento de quem está iniciando os estudos de desenvolvimento de sites.
